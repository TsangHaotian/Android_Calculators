# 简易安卓计算器 ➕➖✖️➗

![GitHub stars](https://img.shields.io/badge/版本-1.0.0-green)
![最小SDK](https://img.shields.io/badge/Android-5.0%2B-blue)

一个专注于四则运算的轻量级计算器应用，适合基础计算需求

## 📱 基础功能

### 核心运算
- **加法**：`12 + 5 = 17`
- **减法**：`20 - 8 = 12`
- **乘法**：`6 × 7 = 42`
- **除法**：`15 ÷ 3 = 5`

### 实用特性
- 实时计算显示
- 退格键修正输入
- 一键清空结果
- 连续计算功能
- 简洁Material设计
